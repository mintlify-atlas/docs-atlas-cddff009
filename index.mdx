## Get started

<Steps>
  <Step title="Download TiXL">
    Grab the latest installer from the [GitHub releases page](https://github.com/tixl3d/tixl/releases) and run it.
  </Step>
  <Step title="Open the editor">
    Launch TiXL. The editor opens with an empty project. You'll see the graph canvas, output window, and parameter panel.
  </Step>
  <Step title="Add your first operator">
    Right-click the graph canvas and search for an operator — try `DrawMesh` or `Gradient`. Connect it to the output.
  </Step>
  <Step title="Animate a parameter">
    Right-click any parameter in the **Parameter** panel and choose **Add keyframe**. Scrub the timeline to animate it.
  </Step>
</Steps>
