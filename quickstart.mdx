---
title: "Quickstart"
description: "Create your first animated scene in TiXL in under five minutes."
---

This guide walks you from a blank canvas to a working animated scene. By the end you will have added operators, connected them, animated a parameter, and previewed the result.

<Steps>

<Step title="Launch TiXL and learn the interface">

Open TiXL from the Start Menu or desktop shortcut. After the splash screen closes, you will see the default editor layout with four main areas:

| Area | What it does |
|---|---|
| **Graph canvas** | The main workspace. Operators appear as nodes; connections are wires between them. |
| **Output window** | Shows the rendered frame from the selected operator in real time. |
| **Parameter panel** | Displays the inputs for the currently selected operator. |
| **Timeline** | A horizontal ruler where you scrub playback time and manage keyframes. |

You can drag panel borders to resize areas, and float or dock panels using the title bar context menu.

</Step>

<Step title="Add your first operator">

The operator library contains all built-in nodes — geometry generators, image effects, math utilities, I/O nodes, and more.

1. **Right-click** anywhere on the empty graph canvas to open the operator search popup.
2. Type part of an operator name, for example `Sphere` or `Noise`.
3. Click the operator in the results list, or press **Enter** to place the top result.

The operator appears on the canvas as a node with input slots on the left and an output slot on the right.

<Tip>
Press **Ctrl+Enter** (or right-click the canvas) to open the symbol library at any time. You can browse by category or search by name.
</Tip>

</Step>

<Step title="Connect operators">

Operators pass data to each other through typed slots. An output slot on one operator connects to an input slot on another.

1. Hover over the **output slot** (right side) of an operator until the cursor changes.
2. Click and drag a wire to an **input slot** (left side) of another operator.
3. Release the mouse to complete the connection.

The types of the slots must be compatible. Incompatible connections are shown in red and will not evaluate.

To disconnect a wire, right-click it and choose **Delete**, or click the input slot and drag the wire away.

</Step>

<Step title="Adjust parameters in the Parameter panel">

Select an operator by clicking it. Its inputs appear in the **Parameter panel** on the right.

- **Drag** a numeric field left or right to change the value continuously.
- **Double-click** a numeric field to type an exact value, then press **Enter**.
- **Right-click** a parameter to access reset, copy, paste, and keyframe options.

Changes take effect immediately and the Output window updates in real time.

</Step>

<Step title="Add keyframe animation">

Any parameter can be animated over time using keyframes.

1. Move the playhead in the **Timeline** to the time where you want the first keyframe.
2. In the **Parameter panel**, right-click the parameter you want to animate and choose **Add keyframe**.
3. Move the playhead to a different time.
4. Change the parameter value — a new keyframe is created automatically.
5. Press **Space** to play back the timeline and see the animation.

Scrub the playhead by clicking and dragging in the Timeline ruler. The Output window updates with every frame.

</Step>

<Step title="Preview your output">

The **Output window** renders the currently selected operator's output continuously.

- Click any operator on the graph canvas to switch the preview to that operator.
- Use the resolution controls in the Output window toolbar to match your target resolution.
- A secondary viewer window (`TiXL Viewer`) opens automatically at startup for an undocked full-resolution preview.

When you are happy with the result, save your project with **Ctrl+S**.

<Note>
Projects are saved as `.t3` files. A `.t3` file is a JSON document that stores all operator graphs, connections, parameter values, and animation keyframes. Keep your project files alongside any external assets (audio, textures, videos) they reference.
</Note>

</Step>

</Steps>
